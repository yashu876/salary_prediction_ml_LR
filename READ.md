# Salary Prediction using Linear Regression

# Project Overview
This project predicts salaries based on factors like age, experience, education, and job title using a Linear Regression model.

# Dataset
- Rows: 6705
- Columns: 6
- Features include:
  - Age
  - Gender
  - Education Level
  - Job Title
  - Years of Experience

# Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

# Project Workflow
1. Data preprocessing
2. Handling missing values
3. One-hot encoding for categorical variables
4. Train-test split
5. Model training using Linear Regression
6. Model evaluation using:
   - MSE
   - RMSE
   - R² Score

# Model Performance

Example results:

- MSE: 387779529
- RMSE: 19692
- R² Score: 0.85
